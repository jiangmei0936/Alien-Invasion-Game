# Alien-Invasion-Game
C# 开发的休闲射击小游戏

## 项目简介
基于 WinForms 开发的休闲射击小游戏，实现玩家操控、子弹发射、碰撞检测、音效播放等核心玩法。

## 技术栈
- 开发语言：C#
- 开发框架：WinForms

## 核心功能
- 玩家移动与射击控制
- 敌人生成与移动逻辑
- 子弹碰撞与消灭判定
- 音效与分数系统

## 运行说明
1. 使用 Visual Studio 打开 `AIG.csproj` 文件
2. 编译并运行项目即可
